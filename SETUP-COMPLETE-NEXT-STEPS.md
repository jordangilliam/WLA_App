## Technical Terms Explained

**Node.js** = Software that lets you run JavaScript on your computer (not just in browsers)

**npm** = Tool that downloads and installs code packages automatically

**Supabase** = A database service (like Google Sheets but for apps, and much more powerful)

**API Route** = A file that handles requests for data (like "get all classes" or "create new student")

**Dependencies** = Other people's code that your app uses (so you don't have to write everything from scratch)

**Portable** = Software that runs without installation (good for locked-down computers)
