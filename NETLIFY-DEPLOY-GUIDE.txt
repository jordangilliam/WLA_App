==============================================

STEP 1: Sign Up / Sign In to Netlify
-------------------------------------
1. Go to: https://netlify.com

2. Click "Sign up" or "Log in"

3. Choose "Sign up with GitHub"

4. Authorize Netlify to access your GitHub


STEP 2: Import Your Project
----------------------------
1. Click "Add new site" button (or "Import from Git")

2. Choose "Import from Git"

3. Select "GitHub"

4. Search for or select: jordangilliam/WLA_App

5. Click on the repository


STEP 3: Configure Build Settings
---------------------------------
Netlify should auto-detect Next.js:

- Branch: main
- Build command: npm run build
- Publish directory: .next

Click "Show advanced" or "Add environment variables"
